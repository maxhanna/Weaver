FAQ
How do I get started with Maestro?

Maestro is an agentic kanban system that helps you manage projects with AI assistance. To get started, create a new project or select an existing one from the project picker.

What is the purpose of the agent activity log?

The agent activity log shows all the actions taken by the AI agent during project management. It helps you track what the agent is doing and why.

How do I add tasks to my project?

You can add tasks by using the 'Add Task' button in the kanban board or by asking the AI assistant to create tasks for you.
