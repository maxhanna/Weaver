❓ FAQ

What is Maestro?

Maestro is an advanced agentic backend system that enables AI-driven automation through intelligent orchestration of multiple tools and pipelines.

How does it work?

Maestro allows you to define projects and use AI agents to perform tasks like code editing, file management, and command execution based on natural language instructions.

What are its key features?

  • AI-powered task automation
  • Intelligent file and project management
  • Terminal integration for command execution
  • Customizable tool hints for better context
  • Real-time agent activity logging

How do I get started?

Configure your project in the settings panel, then select a card and press "Start" to begin an AI agent session.
